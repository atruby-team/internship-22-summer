### 2. Rule khi tạo branch và commit
- Mỗi nhánh các bạn đặt tên theo cú pháp sau: `feature/mvn-hauphan-<name-baitap>`
- Mỗi commit đặt tên có ý nghĩa và có comment nội dung bên trong
